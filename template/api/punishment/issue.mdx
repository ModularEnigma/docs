<TabItem value="first">

```json

    "playerUsername": "shadowolfyt",
    "staffUsername": :"Camemes",
    "platform": "server",
    "type": "kick",
    "length": "5d",
    "reason": "Bad System Admin",
}
```

## Response

```json
{
    "success": true
}
```

shadowolfyt has been kicked from the server for 5d by Camemes for being a 'Bad
System Admin'.
